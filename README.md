# Proyek-Akhir-Sturuktur-Data-dan-Algoritma---Bubbles-Breeze-B-b-
Bubbles &amp; Breeze (B&amp;B) adalah aplikasi laundry berbasis web yang memudahkan pengguna memesan layanan laundry. Menggunakan struktur data list dan queue, serta algoritma Quick Sort dan Linear Search untuk efisiensi pengelolaan pesanan. Dibangun dengan Python dan Tkinter untuk kenyamanan pengguna.
## Fitur Utama:
- Pencarian layanan laundry menggunakan **Linear Search**
- Pengurutan data layanan dengan **Quick Sort**
- Pengelolaan pesanan dengan struktur data **Queue**
## Teknologi yang Digunakan:
- Python
- Tkinter
- Pandas
- Geopy (untuk perhitungan jarak)
